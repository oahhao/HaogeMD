# 南下互认基金系统调整方案

## 一、用需

### 1.1 业务背景

南下互认基金（即内地互认基金）是指依照内地法律注册、经香港证监会认可后，在香港市场向公众销售的内地公募基金。

**准入门槛**：
- 内地证监会注册公募，成立满 1 年
- 基金净资产 ≥ 2 亿元人民币
- 客地（香港）销售份额上限：2025 年新规由 50% 放宽至 80%
- 管理人、风控、信息披露同时满足两地监管标准

**产品特征**：
- 中登TA产品
- 属于 H类子份额
- 母产品可以是中登TA产品，也可以不是
- 母产品原有份额转为 A 类子份额

**支持的业务**：申购、赎回、分红（其他业务中登暂不支持）

**中登不负责交收**。

### 1.2 参与方与账户

| 角色 | 账户 | 说明 |
|------|------|------|
| 香港代销机构 | 销售机构销售账户 | 在港开立 |
| 香港代理人 | 内地基金代销账户（跨境交收归集专户） | 在港开立（FTN/OSA），部分银行要求分上下行账户 |
| 内地管理人 | 南下互认基金募集资金专用账户 | 管理人通过托管人开立，按产品维度，属境内普通账户（非NRA） |
| 内地托管人 | 托管账户 | 基础托管账户 |

### 1.3 资金路径

**文字描述**：

```txt
■ 申购（上行）

  香港个人投资者
    ↓
  香港销售机构销售账户（在港）
    ↓ ① 香港本地转账（香港内部）
  → 香港代理人内地基金代销账户/跨境交收归集专户（在港）
    ↓ ② 跨境人民币（跨越港澳↔内地）
  → 南下募集资金专用账户（内地管理人境内账户）
    ↓ ③ 境内人民币（内地内部）
  → 托管账户

■ 赎回/分红（下行）

  托管账户
    ↓ ① 境内人民币（内地内部）
  → 南下募集资金专用账户（内地管理人境内账户）
    ↓ ② 跨境人民币（跨越港澳↔内地）
  → 香港代理人内地基金代销账户/跨境交收归集专户（在港）
    ↓ ③ 香港本地转账（香港内部）
  → 香港销售机构销售账户
    ↓
  香港个人投资者
```

**交收路径图**：

```mermaid
flowchart LR
    INV1["香港个人投资者"] -->|申购| HK_SALE1["香港销售机构销售账户"]
    HK_SALE1 -->|① 香港本地转账| AGT1["内地基金代销账户<br/>跨境交收归集专户（在港）"]
    AGT1 -->|② 跨境人民币| COLLECT1["南下募集资金专用账户<br/>（境内，按产品开立）"]
    COLLECT1 -->|③ 境内人民币| CUS1["托管账户（内地托管行）"]
    CUS1 --> CUS2["托管账户（内地托管行）"]
    CUS2 -->|① 境内人民币| COLLECT2["南下募集资金专用账户<br/>（境内，按产品开立）"]
    COLLECT2 -->|② 跨境人民币| AGT2["内地基金代销账户<br/>跨境交收归集专户（在港）"]
    AGT2 -->|③ 香港本地转账| HK_SALE2["香港销售机构销售账户"]
    HK_SALE2 -->|赎回/分红| INV2["香港个人投资者"]
```

### 1.4 结算规则

| 规则 | 说明 |
|------|------|
| 与托管户 | 正常轧差交收 |
| 与香港代理人 | 不轧差，赎回分红可合并 |
| 日历 | 确认日、交收日均按共同工作日，无递延日规则 |
| 划款方式 | 银行对南下募集账户有特殊管理要求，无法直连划款，只能查询，出入账手工操作 |

### 1.5 费用处理

| 费用 | 规则 |
|------|------|
| 申购费 | 100% 归香港代销机构，不交收 |
| 赎回费 | 100% 归基金资产，不交收 |
| 销售服务费 | 市场存量产品暂无 |

## 二、软需

### 2.1 基础信息维护

#### 2.1.1 数据字典维护

| 字典 | 编码 | 名称 | 说明 |
|------|------|------|------|
| `DEP_KIND` | **待定** | 香港代理人 | 机构类别 |
| `BANK_ACC_TYPE` | **待定** | 南下互认基金募集资金专用账户 | 银行账户类型 |
| `BANK_ACC_TYPE` | **待定** | 内地基金代销账户（跨境交收归集专户） | 银行账户类型 |
| `ACC_TYPE` | **待定** | 南下互认基金募集资金专用账户  | 账户用途（绑定募集户） |
| `ACC_TYPE` | **待定** | 内地基金代销账户（跨境交收归集专户） | 账户用途（绑定代销户） |
| `INSTR_TYPE` | **待定** | 南下跨境人民币划款指令 | 指令类型 |
| `ROUTE_TEMPLATE_TYPE` | **待定**（如 `22`） | 南下互认基金模板 | 交收路径模板类型 |



#### 2.1.2 机构维护

| 功能 | 内容 |
|------|------|
| 维护表 | `EA_FT2.FT_DEPT_INFO` |
| 机构类别 | `DEP_KIND` = 香港代理人（新增字典值） |
| 维护内容 | 机构名称、英文名、国家/地区、地址、联系人、联系方式等 |
| 产品关联 | 维护产品时从香港代理人列表中选择 |

> 香港代理人≠香港管理人。代理人是销售服务中介（类似代销），管理人是基金管理方。

#### 2.1.3 账户维护

**南下互认基金募集资金专用账户**：

| 项目 | 内容 |
|------|------|
| 维护表 | `EA_FT2.FT_BANK_ACC_INFO` |
| 账户类型 | `BANK_ACC_TYPE` = 南下募集资金专用账户（新增字典值） |
| 账户用途 | 通过"账户用途维护"绑定到产品 |
| 账户性质 | 境内普通账户 |
| 开立方式 | 管理人通过托管人开立，按产品维度，不共用 |

**内地基金代销账户（跨境交收归集专户）**：

| 项目 | 内容 |
|------|------|
| 维护表 | `EA_FT2.FT_BANK_ACC_INFO` |
| 账户类型 | `BANK_ACC_TYPE` = 内地基金代销账户（跨境交收归集专户）（新增字典值） |
| 境外标识 | `IS_EXTERNAL_ACC = '1'` |
| 上下行账户 | 支持区分上行账户和下行账户 |
| 账户性质 | 在港开立（FTN/OSA），境外账户 |

#### 2.1.4 产品维护

| 功能 | 说明 |
|------|------|
| 南下标记 | 子产品新增「南下互认基金」复选框（`IS_SNMR_FUND='1'`） |

| 绑定募集户 | 选择已维护的南下募集资金专用账户 |
| 绑定代销户 | 选择已维护的内地基金代销账户（跨境交收归集专户） |



#### 2.1.5 账户用途维护

| 功能 | 说明 |

| 维护入口 | 系统已有菜单"账户用途维护" |
| 绑定表 | `EA_FOCS.OF_FUND_BANK_ACC` |
| 南下募集用途 | 产品 → 南下募集资金专用账户，`ACC_TYPE`=南下募集收款用途 |
| 南下代销用途 | 产品 → 内地基金代销账户/跨境交收归集专户，`ACC_TYPE`=南下代销付款用途 |



#### 2.1.6 交收指令维护

| 功能 | 说明 |
|------|------|
| 指令类型 | 新增：香港代理人交收指令（`INSTR_TYPE`新编码） |
| 轧差规则 | 托管户：正常轧差交收 |
| | 香港代理人：不能跨产品，不轧差（逐笔或合并划付） |
| 合并规则 | 不能跨产品，赎回和分红可合并划付给香港代理人 |

#### 2.1.7 交收路径维护

| 功能 | 说明 |
|------|------|

| 备注 | 南下模板不涉及北上的 `IS_MC_SETTLE`（港股通募集专户标记） |



#### 2.1.8 交收日历维护

| 功能 | 说明 |
|------|------|
| 日历模板 | 南下互认基金模板绑定互认业务日历 |

## 三、与北上互认的关键差异

| 维度 | 北上（已实现） | 南下（待实现） |
|------|---------------|---------------|
| 产品标识 | `IS_NSMR_FUND` | `IS_SNMR_FUND` |
| 模板类型 | `21` | **待定**（如`22`） |
| 基金类型 | 香港基金 | 内地基金H类子份额 |
| 募集账户 | 香港管理人NRA账户（境外） | 内地管理人境内普通账户 |
| 对手方 | 香港管理人（NRA托管户） | **香港代理人**（代销户） |
| 递延日 | 有 | **无** |
| 划款通道 | NRA划款接口（`IS_NRA_FT`） | **南下独立通道**（`IS_SNMR_FT`） |
| 资金路径 | 经清算总账户 | **直通募集户** |
| 中登交收 | 部分场景支持 | 中登不负责交收 |
| 银行直连 | NRA国际划款接口 | **无法直连，手工出入账** |
